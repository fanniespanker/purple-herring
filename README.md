# Purple Herring

Purple Herring is the public C4 language and tooling ecosystem.

C4 is the Contextual Compositional Concept Calculus: a compact graph-native language for semantic relation structures.

## Start here

- `specs/c4/c4_specification.md` — the technical C4 specification.
- `specs/c4/c4_for_non_pescatarians.md` — a plain-language companion guide to C4.
- `specs/cathedral/cathedral_specification.md` — public graph execution and patch semantics.
- `specs/herring-bones/herring_bones_specification.md` — reusable scaffolds, bones, and skeletons.
- `specs/librarium/` — public authoring and markup specifications.

## Main components

- **C4** — the core language for relation-fish, schools, resources, bindings, and context grounding.
- **SARDINE** — parser, canonicalizer, formatter, validator, and toolchain.
- **Herring Bones** — reusable semantic scaffolds, relation skeletons, validation profiles, and template libraries.
- **CAThedral** — public graph execution and patch semantics.
- **LIBRARIUM** — public authoring and markup specifications such as CNML, PÂTE, and CiNnaMoN.
